# EcoTech - Admin Panel (Angular)

Painel administrativo desenvolvido em **Angular** para gerenciamento e monitoramento do sistema **EcoTech**.  
O objetivo é fornecer aos administradores uma visão centralizada sobre **usuários, coletas de e-lixo, pontos e estatísticas ambientais**, integrando-se diretamente à **API EcoTech (Spring Boot)**.  

Este módulo complementa o **aplicativo mobile (Flutter)**, formando uma solução completa no contexto da **Sociedade 5.0**.

![Angular](https://img.shields.io/badge/Angular-18-red) ![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue) ![License](https://img.shields.io/badge/License-Academic-lightgrey)

---

## ✨ Funcionalidades

- **Login administrativo** (integração com API de autenticação JWT)  
- **Gerenciamento de usuários** (listagem, edição e exclusão)  
- **Visualização de pontos e recompensas**  
- **Monitoramento do impacto ambiental** (estatísticas integradas)  
- **Integração direta com a API EcoTech (Spring Boot)**  

---

## 🛠️ Tecnologias Utilizadas

- **Angular 18**  
- **TypeScript 5.x**  
- **RxJS**  
- **Angular Services** (consumo de API REST)  
- **HTML5 / CSS3 (Material Design)**  
- **Integração com Spring Boot API**  

---

## 🚀 Como Executar o Projeto

```bash
# Clone o repositório
git clone https://github.com/ThiagoAlvesFeitosa/ecotech_angular_admin.git

# Acesse a pasta
cd ecotech_angular_admin

# Instale as dependências
npm install

# Execute a aplicação em modo de desenvolvimento
ng serve
````

Acesse no navegador:
👉 [http://localhost:4200](http://localhost:4200)
